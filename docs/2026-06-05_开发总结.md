## 五、架构图

```
用户输入 → [命令处理 / 图片检测 / 粘贴检测]
              │
              ▼
          上下文管理 ← sanitize + smart_trim (LLM压缩)
              │
              ▼
          API 调用 → DeepSeek / 本地模型
              │     reasoning_content 按规范回传
              │     image_url 按模型能力发送/剥离
              ▼
         工具执行 → 46 个工具
              │    recall_conversation (FTS5搜索)
              │    update_profile (用户画像)
              │    execute_python (持久化REPL)
              │    analyze_image / capture_camera (视觉)
              ▼
         记忆存储 → SQLite + FTS5 (原文永久保留)
              │    meta 表 (滚动摘要 + 用户画像)
              ▼
         输出 → Rich Markdown + TTS
```

## 六、使用建议

1. **主模型用 DeepSeek**（工具调用精准，128K 上下文）
2. **视觉模型用本地 Gemma**（`vision_base_url` 指向 `192.168.8.200:1234`）
3. **设 `user_city`** 免去每次天气查询问城市
4. **`/profile add`** 让模型记住你的偏好
5. **代码写到 `output/`**，临时文件自动进 `temp/`
