## 一、Provider 适配器层

借鉴 Proma 的 ProviderAdapter 架构：

- **抽象基类** (base.py)：统一 StreamRequestInput → ProviderRequest 转换
- **注册表** (registry.py)：Map 模式 + 自动检测 provider 类型
- **4 种适配器**：
  - DeepSeekAdapter — 思考模式 + 缓存命中 + reasoning_effort
  - OpenAICompatAdapter — OpenAI 兼容协议
  - AnthropicCompatAdapter — Anthropic Messages API（解锁 Claude）
  - LocalAdapter — Ollama / LM Studio

```
from orca_code.providers import get_adapter
adapter = get_adapter("deepseek")
request = adapter.build_stream_request(input)
```

---

## 二、工具系统重构

借鉴 Claude Code 的 Tool 类继承架构：

- **Tool 基类**：ABC + JSON Schema 声明 + RiskLevel 自标注
- **ToolRegistry**：中心化注册表 + execute + schema 导出
- **Bridge**：63 遗留工具无缝桥接 + EventBus 接线
- **53 个 Tool 子类**：8 核心 + 5 Web + 6 Dev + 6 Office + 12 Automation + 14 Extended + 4 Task

每个 Tool 类自动获得：JSON Schema 验证、EventBus 生命周期事件、RiskLevel 权限检查

## 三、核心抽象层

### 错误分类 (core/errors.py)

借鉴 Proma 的 error-patterns.ts：

```
7 类错误：Network | Auth | RateLimit | Model | Tool | Permission | Internal
classify_error() → (ErrorCategory, is_retryable)
execute_with_retry() → 指数退避自动重试
```

### 事件总线 (core/event_bus.py)

借鉴 Proma 的 AgentEventBus：

```
23 种事件类型：
  Tool: tool_start, tool_result, tool_error
  Agent: agent_start, agent_done, agent_error
  Stream: stream_chunk, stream_done, stream_error
  Permission: permission_request, permission_result
  Session: session_start, session_end, turn_start, turn_end
  Memory: memory_saved, memory_searched
  Task: task_created, task_updated, task_deleted

发布-订阅 + 线程安全 + 单例 + 装饰器注册
```

## 四、基础设施

### 配置拆分
- **config_loader.py**：纯配置加载，零副作用
- **config.py**：薄兼容层，client/mem_mgr 懒初始化
- ~~星号导入~~ → `__getattr__` 按需懒加载（import 143x 加速）

### 特征开关 (feature_flags.py)

借鉴 Claude Code 的 feature() 编译开关：

```
15 个开关：ENABLE_LSP, ENABLE_GUI_AUTO, ENABLE_BROWSER,
          ENABLE_VOICE, ENABLE_MCP, ENABLE_SUBAGENTS, ...
环境变量覆盖：ORCA_ENABLE_BROWSER=1
```

### 指标收集 (metrics.py)

```
工具耗时 p50/p95/p99 + 错误率
EventBus 自动订阅 TOOL_START/RESULT/ERROR
get_metrics_collector().get_summary()
```

### 文件日志 (file_logger.py)

```
JSONL 结构化日志 → logs/orca-YYYY-MM-DD.jsonl
EventBus 自动订阅 + 按日切割
```
