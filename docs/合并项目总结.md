> 日期：2026-06-07 | 版本：v4.0.0

## 做了什么

以 Orca Code v3 为底座，将 VulnClaw 的渗透测试能力作为安全领域模块集成进来。
合并后产品既能做通用桌面办公自动化，也能做专业授权渗透测试。

## 对本项目（Orca Code v3）的改进
